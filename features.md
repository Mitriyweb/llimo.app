# LLiMo features

1. Опрацювання тексту, аудіо, фото, відео, для тих моделей, які підтримують через `ai-sdk`.
1. Release - для розбивки задач на тести і їхнє виконання.
1. Можливість запускати `llimo` в одному проєкті у різних терміналах, таким чином можна працювати з різними активним чатами. Тобто інформація про активний чат має бути ізольованою для тернінала. Зараз ця інформація зберігається у `chat/current` з ID активного чату.
